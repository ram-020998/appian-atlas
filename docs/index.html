What is Appian Atlas?

Appian Atlas is a comprehensive system that transforms Appian application packages into structured, AI-queryable knowledge bases. It combines powerful parsing technology with an intelligent MCP server and persona-specific AI assistants to help developers, product owners, and UX designers understand and work with Appian applications more effectively.

Quick Start

1. Install the MCP Server

# Create virtual environment
python3 -m venv ~/.venvs/appian-atlas
source ~/.venvs/appian-atlas/bin/activate

# Upgrade pip
pip install --upgrade pip

# Install appian-atlas
pip install "git+https://github.com/ram-020998/gam-knowledge-base.git#subdirectory=mcp_server"

2. Configure GitHub Token

Generate a GitHub Personal Access Token and add it to your Power Config in Kiro IDE.

3. Install a Power

In Kiro IDE, go to the Powers tab and add one of these URLs:

# For Developers
https://github.com/ram-020998/power-appian-atlas-developer

# For Product Owners
https://github.com/ram-020998/power-appian-atlas-product-owner

# For UX Designers
https://github.com/ram-020998/power-appian-atlas-ux-designer

4. Start Asking Questions

"What applications are available?"
"Give me an overview of SourceSelection"
"How does the Add Vendors action work?"
"What depends on the vendor validation logic?"
